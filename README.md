# Meu projeto Pokédex

Projeto do bootcamp da DIO, utilizando html, css e javascript, requisitando dados da API PokéAPI, para criar uma pokédex com diversos pokemons, e uma tela para suas respectivas informações.

https://pokeapi.co/
